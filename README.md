Identidade Visual
A identidade visual do CACA foi concebida para transmitir profissionalismo académico e confiança na área da saúde, integrando elementos regionais dos Açores.

Paleta de Cores
Azul Primário (#1976D2 / #0D47A1): Representa confiança, saúde e o oceano Atlântico.

Verde Secundário (#43A047 / #2E7D32): Simboliza vitalidade e esperança.

Destaques: Cores vibrantes (laranja, roxo, vermelho) para diferenciar as áreas de investigação.

Tipografia
Display: Playfair Display (Serifada para títulos institucionais).

Corpo: Source Sans 3 (Sans-serif para legibilidade).

Estilo Geral
Design limpo e profissional com elementos visuais inspirados nas paisagens açorianas.

Layout moderno utilizando CSS Grid e Flexbox.

Animações suaves e estados hover refinados para melhorar a experiência do utilizador.

Evolução: Wireframe → Mockup → Implementação
Wireframes: Foco na estrutura e hierarquia da informação (cabeçalho, hero, 5 cards de investigação, parceiros, oportunidades e contactos).

Mockup: Definição da paleta final, espaçamentos e elementos decorativos (gradientes e padrões subtis).

Implementação Final: Tradução fiel para HTML5 e CSS3.

Melhorias: Otimização de contraste e ajustes de espaçamento.

Alterações principais: Simplificação gráfica para performance e ajuste de fontes para legibilidade móvel.

Acessibilidade e Responsividade
Acessibilidade (WCAG)
Contraste: Garantia de ratio mínimo de 4.5:1 para texto normal.

Semântica: Uso de tags apropriadas como <header>, <nav>, <section>, <article> e <footer>.

Imagens: Atributos alt em todas as imagens relevantes.

Navegação: Otimizada para teclado com estados :focus-visible visíveis.

Formulários: Labels associados a todos os campos de entrada.

Responsividade
Abordagem: Estratégia adaptativa com breakpoints em 480px, 768px e 1024px.

Layout Flexível: Transição de CSS Grid multi-coluna (desktop) para colunas únicas (mobile).

Menu: Reorganização vertical para ecrãs pequenos.

Dispositivos Testados: Desktop (1920px), Tablet (768px) e Smartphone (375px).
