## Identidade Visual
A identidade visual do **CACA** foi concebida para transmitir profissionalismo académico e confiança na área da saúde, integrando elementos regionais dos Açores.

###  Paleta de Cores
* **Azul primário (#1976D2 / #0D47A1):** Representa confiança, saúde e o oceano Atlântico que rodeia as ilhas.
* **Verde secundário (#43A047 / #2E7D32):** Simboliza saúde, vitalidade e esperança.
* **Cores de destaque (laranja, roxo, vermelho):** Utilizadas para diferenciar áreas de investigação.

### Tipografia
* **Display:** Playfair Display (usada em títulos institucionais).
* **Corpo:** Source Sans 3 (otimizada para leitura de conteúdo).

###  Estilo Geral
* Design limpo e profissional com elementos visuais inspirados nas paisagens açorianas.
* Layout moderno utilizando **CSS Grid** e **Flexbox**.
* Animações suaves para melhorar a experiência do utilizador.
* Ícones minimalistas e elementos gráficos que reforçam a identidade regional.

## Evolução: Wireframe → Mockup → Implementação

### Wireframes
Focaram-se na estrutura e hierarquia da informação, definindo a disposição de cabeçalho, hero, áreas de investigação (5 cards), parceiros, oportunidades e contactos. Foram criadas versões desktop e mobile para garantir responsividade.

### Mockup
Adicionou detalhes visuais completos: paleta de cores final, tipografia específica, espaçamentos refinados e elementos decorativos como gradientes e padrões subtis. O mockup serviu como referência visual precisa para a implementação.

### Implementação Final
Traduziu fielmente o mockup para **HTML5** e **CSS3**, mantendo a estrutura dos wireframes.
* **Melhorias:** Otimização de contraste para acessibilidade, ajustes de espaçamento para diferentes dispositivos e refinamento de estados hover.
* **Alterações principais:** Simplificação de alguns elementos gráficos para melhor performance web, ajuste de tamanhos de fonte para melhor legibilidade em dispositivos móveis e adição de estados de foco mais evidentes.

##  Acessibilidade e Responsividade

### Acessibilidade (WCAG)
* **Contraste de cores:** Testado para garantir ratio mínimo 4.5:1 (texto normal) e 3:1 (texto grande).
* **Atributos alt:** Incluídos em todas as imagens relevantes.
* **Labels:** Associados a todos os campos de formulário.
* **Navegação por teclado:** Otimizada com estados `:focus-visible` visíveis.
* **Hierarquia semântica:** Uso de tags HTML5 apropriadas (header, nav, section, article, footer).

### Responsividade
* **Mobile-first:** Breakpoints definidos em 480px, 768px e 1024px.
* **Layout Flexível:** Uso de CSS Grid que se adapta de múltiplas colunas (desktop) para colunas únicas (mobile).
* **Menu de Navegação:** Reorganiza-se verticalmente em ecrãs pequenos.
* **Tipografia Responsiva:** Ajustes de tamanho para legibilidade em todos os dispositivos.
* **Testado em:** Desktop (1920px), Tablet (768px) e Smartphone (375px).
